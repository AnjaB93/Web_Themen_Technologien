Wie ist ein HTML-Dokument aufgebaut?

HTML steht für Hypertext Markup Language un dist die Standard-Markup-Sprache. Hierdurch wird die Struktur einer WEbiste beschrieben, da ein HTML-Dokument aus vielen verschiedenen Elementen besteht, welche dem Browser sagt wie er die Inhalte darstellen soll. Sprich die Elemente bezeichnen Inhalte, sodass genau gekennzeichnet ist, was ist eine Überschrift, ein Absatz oder ein Bild, etc.

Ein einfaches HTML-Dokument ist folgendermaßen aufgebaut:

            
  <!DOCTYPE html> 
  <html>
  <head>
  <title>Website Titel</title>
  </head>
  <body>
           
  <h1>Überschrift</h1>
  <p>Dies ist ein Paragraph.</p>
                
  </body>
  </html>
